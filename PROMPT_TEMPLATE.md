# Промпт для саммари в Claude

## Как использовать

1. Раз в месяц GitHub автоматически соберёт данные (15-го числа)
2. Придёт уведомление (GitHub Issue на email)
3. Зайди в репо → `docs/for_claude.md` → скачай файл
4. Открой новый чат с Claude (claude.ai)
5. Прикрепи скачанный файл
6. Скопируй промпт ниже и отправь

## Промпт (скопируй всё между линиями)

---

Я прикрепила файл с отфильтрованными экспертными материалами за этот месяц. Там же — краткая база знаний с ключевыми тезисами Далио, Турчина, Хоува, Талеба и ITR Economics. Теории пересказывать не надо, я их знаю.

Напиши мне саммари. Обо мне: живу в Брно (Чехия), работаю в IT, владею квартирой в Украине.

ВАЖНО: Некоторые статьи в файле пришли как тизеры/обрезанные тексты. Если видишь, что текст обрезан, а статья важная — используй веб-поиск, сходи по URL из файла и прочитай полный текст. Особенно это касается ITR Economics, ECB Blog и аналитических статей.

Также используй веб-поиск, чтобы дополнить данные по Чехии:
- Текущая ставка ČNB
- Последние данные по инфляции в ЧР
- Курс CZK/EUR
- Средняя ставка по ипотеке в Чехии
- Цены на электричество и газ в Чехии

Правила:
- Только конкретные факты с цифрами
- Если данных нет — не пиши раздел, одна строка "Новых данных нет"
- Не пиши "стоит мониторить", "рассмотри возможность" — только конкретные действия
- Инвестиционные советы: называй конкретные инструменты (ETF тикеры, чешские облигации ČOAZ, платформы XTB/Degiro), секторы и компании. Говори, что покупать и что НЕ покупать
- Ближний Восток — только если влияет на цены энергии в Чехии, максимум один абзац
- Каждый пункт с новой строки

Структура:

**Главное за месяц** — 3-5 фактов с цифрами

**Чехия и мои финансы** — это главный раздел (50% текста):
- Ставка ČNB, инфляция, курс CZK/EUR, энергия, ипотека — только те пункты, по которым есть цифры
- Отдельно: что делать с деньгами прямо сейчас (банки, инструменты, что покупать/не покупать)

**Европа** — решения ЕЦБ, ставки → что это значит для меня

**Украина** — только конкретные изменения для владельца квартиры (законы, курс гривны, компенсации, цены). Если ничего — одна строка

**Что делать (3-5 пунктов)** — формат: **[факт]** → [действие на этой неделе]

Если чего-то не хватает в данных или что-то непонятно — спроси, я дополню.
